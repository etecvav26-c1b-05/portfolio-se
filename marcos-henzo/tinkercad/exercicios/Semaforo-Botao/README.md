# 🚦 Semáforo Duplo com Travessia de Pedestres

## 📖 Sobre o Projeto

Este projeto consiste na implementação de um sistema de **semáforo duplo controlado por microcontrolador**, simulando o funcionamento de um cruzamento com passagem de pedestres.

O sistema possui dois conjuntos de semáforos para veículos e um semáforo para pedestres. A travessia de pedestres ocorre somente quando o botão é pressionado, tornando o funcionamento mais eficiente e semelhante aos semáforos encontrados em vias urbanas.
