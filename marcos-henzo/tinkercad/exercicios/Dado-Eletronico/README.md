🎲 Dado Eletrônico
📖 Sobre o Projeto

Este projeto consiste na implementação de um dado eletrônico controlado por microcontrolador, simulando o funcionamento de um dado convencional utilizado em jogos.

O sistema utiliza um botão para realizar o lançamento do dado. Quando pressionado, o microcontrolador executa um sorteio aleatório e exibe um número entre 1 e 6, representando as possíveis faces de um dado tradicional.
