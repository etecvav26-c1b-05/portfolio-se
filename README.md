## 📚 Sobre o Repositório

Este repositório reúne atividades, experimentos e projetos desenvolvidos durante a disciplina de **Sistemas Embarcados**.

Aqui documentamos nossa evolução na criação de sistemas capazes de interagir com o ambiente através de sensores, atuadores e microcontroladores, aplicando conceitos de eletrônica e programação.
